# My Coding Journey 🌌

A reflection page about your first 2 days in the program — how you're seeing coding and AI so far. Everything for this project is done **directly in the browser on GitHub** — no downloads, no installs.

## What's in this project

| File | Purpose |
|---|---|
| `index.html` | The page content — your story, your words |
| `style.css` | The gradients, glow, and animations |
| `script.js` | The interactive "reveal my word" button |

## Step 1 — Get your own copy

On the workshop repo page, click **Use this template** (or **Fork** if you don't see that button) at the top right. This creates a copy under your own GitHub account.

## Step 2 — Edit files right on GitHub

You never need to leave the browser:

1. Open your copy of the repo.
2. Click on `index.html`.
3. Click the **pencil icon (Edit)** in the top right of the file view.
4. Type your changes directly into the editor.
5. Scroll down, add a short message describing what you changed, and click **Commit changes**.

Do the same for `style.css` and `script.js` when you're ready to change colors or the interactive word.

> 💡 Want a fuller code editor experience, still 100% in the browser? Press the **`.`** key while viewing your repo on GitHub. This opens **github.dev**, a full editor tab — no install, still just GitHub.

## Step 3 — Add your own photos

1. Take or find a photo you want to use for Day 1 or Day 2.
2. On your repo page, click **Add file → Upload files**.
3. Drag your photo in and commit it.
4. Edit `index.html` and change `day1-placeholder.jpg` (or `day2-placeholder.jpg`) to match your uploaded file's name exactly, including the extension.

## Step 4 — See it live with GitHub Pages

1. Go to **Settings → Pages** in your repo.
2. Under "Build and deployment," set the branch to `main` and folder to `/root`.
3. Click **Save**. After a minute, your page will be live at:
   ```
   https://YOUR-USERNAME.github.io/REPO-NAME/
   ```
4. Every time you commit a change, refresh that link to see it update.

## What to include in your reflection

- **Day 1 & Day 2 sections** — a real moment, win, or struggle from each day
- **"Coding, to me, feels like..."** — your honest one-liner
- **"AI, to me, feels like..."** — your honest one-liner
- **At least one photo** of your own
- **Your own word** in `script.js` for the reveal button

## Make it stunning — ideas to try

The starter already has a shifting gradient title, glowing glass-style cards, and hover effects on the photos. Push it further by editing `style.css`:

- Change the gradient colors in `.gradient-text` and `.hero`
- Try a different `animation` duration or a new `@keyframes` effect
- Add a hover effect to `.thought-card`
- Swap the button gradient colors in the `button` rule

## Need help?

Ask your facilitator, or open an **Issue** on your repo describing what's not working — that's exactly how real developers get support too.
